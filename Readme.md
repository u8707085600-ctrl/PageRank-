# 📋 Kanban Board - Task Management Application

A beautiful, fully-functional Kanban board built with vanilla HTML, CSS, and JavaScript.

## ✨ Features

- 🎯 **Drag-and-Drop Tasks** - Move tasks between columns
- ➕ **Add New Tasks** - Create with title, description, and priority
- 🗑️ **Delete Tasks** - Remove with confirmation
- 📊 **Custom Columns** - Create your own workflow stages
- 💾 **Auto-Save** - Data persists in browser's local storage
- 📱 **Responsive Design** - Works on desktop, tablet, mobile

## 🚀 Getting Started

1. Download or clone this repository
2. Open `kanban-board.html` in your web browser
3. Start organizing your tasks!

No installation, no dependencies, no server required!

## 📝 How to Use

- **Add Task**: Click "+ Add Task" on any column
- **Move Task**: Drag cards between columns
- **Delete Task**: Click "Delete" button on a task
- **Add Column**: Click "+ Add Column" to create custom workflow stages
- **Clear Board**: Click "Clear Board" to reset everything

## 💾 Data Storage

- All data is saved to your browser's local storage
- Your tasks persist even after closing the browser
- Clear your browser cache to reset the board

## 🎨 Customization

Edit `kanban-board.html` to customize:
- Colors and theme
- Default columns
- Sample tasks
- Language and text
